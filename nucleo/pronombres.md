# Pronombres

## Pronombres Personales

| Castellano     | Marroquí |
|----------------|----------|
| Yo             | ana      |
| Tu (masculino) | nta      |
| Tu (femenino)  | nti      |
| El             | huwa     |
| Ella           | hiya     |
| Nosotr@s       | Hna      |
| Ell@s          | huma     |
| Vosotr@s       | ntuma    |
