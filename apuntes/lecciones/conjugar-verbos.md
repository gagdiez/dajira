# Conjugar Verbos

Para conjugar verbos con la persona que la persona que los acciona podemos utilizar la siguiente regla:


| Persona | Prefijo - Sufijo      |
|:--------|:----------------------|
| ana     | `ka``n` \<verbo\>     |
| nta     | `ka``t` \<verbo\>     |
| nti     | `ka``t` \<verbo\> `i` |
| huwa    | `ka``y` \<verbo\>     |
| hiya    | `ka``t` \<verbo\>     |
| Hna     | `ka``n` \<verbo\> `u` |
| huma    | `ka``t` \<verbo\> `u` |
| ntuma   | `ka``y` \<verbo\> `u` |
