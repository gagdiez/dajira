# Estudiar
Infinitivo: `kteb`

| Castellano | Marroquí      |
|:-----------|:--------------|
| Yo         | ana kan`kteb`    |
| Tu (m)     | nta kat`kteb`    |
| Tu (f)     | nti kat`kteb`y   |
| El         | huwa kay`kteb`   |
| Ella       | hiya kat`kteb`   |
| Nosotr@s   | Hna kan`kteb`u   |
| Ustedes    | ntuma kat`kteb`u |
| Ell@s      | huma kay`kteb`u  |
