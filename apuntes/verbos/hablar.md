# Hablar
Infinitivo: `hdar`

| Castellano | Marroquí       |
|:-----------|:---------------|
| Yo hablo   | ana kanhadar   |
| Tu (m)     | nta kanhdar    |
| Tu (f)     | nti kathdar    |
| El         | huwa kayhdari  |
| Ella       | hiya kathdar   |
