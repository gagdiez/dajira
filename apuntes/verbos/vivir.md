# Vivir (Habitar)
Infinitivo: `skun`

| Castellano | Marroquí      |
|:-----------|:--------------|
| Yo         | ana kan`skun`    |
| Tu (m)     | nta kat`skun`    |
| Tu (f)     | nti kat`skun`y   |
| El         | huwa kay`skun`   |
| Ella       | hiya kat`skun`   |
| Nosotr@s   | Hna kan`skun`u   |
| Ustedes    | ntuma kat`skun`u |
| Ell@s      | huma kay`skun`u  |

## Negado

| Castellano  | Marroquí           |
|:------------|:-------------------|
| Yo no       | ana ma kan`skun`sh    |
