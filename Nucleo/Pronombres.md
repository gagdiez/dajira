Dos cosas importantes a destacar:
1. En el marroquí el "tu" distingue masculino de femenino.
2. La `"H"` de "Hna" es fuerte.

| Castellano | Marroquí |
| ---------- | --------- |
| Yo | ana |
| Tu (masculino) | nta |
| Tu (femenino) | nti |
| El | huwa |
| Ella | hiya |
| Nosotr@s | Hna |
| Ell@s | huma |
| Vosotr@s | ntuma |
