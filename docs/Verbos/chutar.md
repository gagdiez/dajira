# Chutar (pegar)
Infinitivo: `drb`

| Castellano | Marroquí        |
|:-----------|:----------------|
| Yo         | ana kan`drb`    |
| Tu (m)     | nta kat`drb`    |
| Tu (f)     | nti kat`drb`i   |
| El         | huwa kay`drb`   |
| Ella       | hiya kat`drb`   |
| Nosotr@s   | Hna kan`drb`u   |
| Ustedes    | ntuma kat`drb`u |
| Ell@s      | huma kay`drb`u  |

### Negado

| Castellano  | Marroquí             |
|:------------|:---------------------|
| Yo no       | ana ma kan`drb`sh    |

| Ella no     | hiya ma kat`drb`sh   |
| Nosotr@s no | Hna ma kan`drb`ush   |
| Ustedes no  | ntuma ma kat`drb`ush |
