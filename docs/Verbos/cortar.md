# Cortar

| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana kan`qta3`    |
| Tu (m)     | nta kat`qta3`    | 
| Tu (f)     | nti kat`qta3`i   |

| Ella       | hiya kat`qta3`   |
