## Htta (7ta)

Htta (también escrito como 7ta) es una palabra con muchas traducciones.

| Marroquí           | Castellano                |
|:-------------------|:--------------------------|
| Htta ana           | yo también                |
| Htta ana la        | yo tampoco                |
| Htta wahd          | nadie                     |
| Htta haja          | ninguna cosa              |
| Httal simana jjaya | hasta la semana que viene |
