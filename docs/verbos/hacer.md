# Hacer
Infinitivo: `dir`

| Castellano | Marroquí        |
|:-----------|:----------------|
| Yo         | ana kan`dir`    |
| Tu (m)     | nta kat`dir`    |
| Tu (f)     | nti kat`dir`i   |
| El         | huwa kay`dir`   |
| Ella       | hiya kat`dir`   |
| Nosotr@s   | Hna kan`dir`u   |
| Ustedes    | ntuma kat`dir`u |
| Ell@s      | huma kay`dir`u  |
