# Chutar (pegar)
Infinitivo: `drb`

| Castellano | Marroquí        |
|:-----------|:----------------|
| Yo         | ana   ka`n` `drb`    |
| Tu (m)     | nta   ka`t` `drb`    |
| Tu (f)     | nti   ka`t` `drb`i   |
| El         | huwa  ka`y` `drb`   |
| Ella       | hiya  ka`t` `drb`   |
| Nosotr@s   | Hna   ka`n` `drb`u   |
| Ustedes    | ntuma ka`t` `drb`u |
| Ell@s      | huma  ka`y` `drb`u  |
