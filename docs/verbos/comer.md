# Comer
Infinitivo: `akul`

| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana kan`akul`    |
| Tu (m)     | nta kat`akul`    | 
| Tu (f)     | nti kat`akul`i   | 
| El         | huwa kay`akul`   | 
| Ella       | hiya kat`akul`   | 
| Nosotr@s   | Hna kan`akul`u   | 
| Ustedes    | ntuma kat`akul`u | 
| Ell@s      | huma kay`akul`u  |
