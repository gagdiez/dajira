# Comer
Infinitivo: `kla`

## Pasado
| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana   `kli`t    |
| Tu (m)     | nta   `kli`t    | 
| Tu (f)     | nti   `kli`ti   | 
| El         | huwa  `kla`   | 
| Ella       | hiya  `kla`t   | 
| Nosotr@s   | Hna   `kli`na   | 
| Ustedes    | ntuma `kli`tu | 
| Ell@s      | huma  `kla`u  | 


## Presente
`Comer` es uno de los dos verbos que cambia su estructura interna en el presente continuo (el otro es `Agarrar`)

| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana kan`akul`    |
| Tu (m)     | nta kat`akul`    | 
| Tu (f)     | nti kat`akul`i   | 
| El         | huwa kay`akul`   | 
| Ella       | hiya kat`akul`   | 
| Nosotr@s   | Hna kan`akul`u   | 
| Ustedes    | ntuma kat`akul`u | 
| Ell@s      | huma kay`akul`u  |
