# Tener

En Darija para denotar posesión se puede utilizar el **adverbio** `3nd`.

| Castellano       | Marroquí       |
|:-----------------|:---------------|
| Yo tengo         | ana 3nd`i`     |
| Tu tienes        | nta/nti 3nd`k` |
| El tiene         | huwa 3nd`u`    |
| Ella tiene       | hiya 3nd`ha`   |
| Nosotr@s tenemos | Hna 3nd`na`    |
| Ustedes tienen   | ntuma 3nd`kum` |
| Ell@s tienen     | huma  3nd`hum` |
